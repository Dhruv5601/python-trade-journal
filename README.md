# 📈 Trade Journal (Python)

A console-based Trade Journal application built in Python to help record, store, and analyze trading performance.

This project was created to strengthen my understanding of object-oriented programming, file handling, exception handling, and Python application design.

---

## Features

- Add new trades through a menu-driven interface
- Store trades persistently in a text file
- Load previously saved trades automatically
- View complete trade history
- Calculate trading statistics including:
  - Total Trades
  - Winning Trades
  - Losing Trades
  - Break-even Trades
  - Win Rate
  - Average Profit
  - Best Trade
  - Worst Trade
- Input validation and exception handling for invalid trade data
